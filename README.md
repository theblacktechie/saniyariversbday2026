# Saniya Rivers Birthday Data Viz
**Happy Birthday #22 · March 4, 2003 · Turning 23**

A Sportico-style interactive stat poster celebrating Saniya Rivers across four career chapters.

## How to Use

Open `saniya-rivers-sportico.html` in any modern browser. No server, no dependencies, no install required.

## Features

- **Four era panels:** High School · College · WNBA Rookie · Unrivaled Vinyl
- **Click any card** to expand deeper stats and an editorial story note
- **Hover glow:** each card glows in its era color on hover
- **Confetti** bursts on load and can be re-triggered three ways:
  - CT Sun button (top right)
  - Unrivaled button (top right)
  - Celebrate button (footer)
- **Fully mobile responsive:** 2-column on small screens, 4-column on desktop
